# DCER Connect

Portal dos Embaixadores do Rei de Mato Grosso do Sul, desenvolvido para centralizar informações, eventos, notícias, materiais e conteúdos institucionais da organização no estado.

O projeto está sendo construído inicialmente como frontend independente. A integração com a API e com o backend será realizada em uma etapa futura.

## Sobre o projeto

O DCER Connect tem como objetivo oferecer uma experiência moderna, acessível e responsiva para:

- Embaixadores do Rei
- Conselheiros
- Lideranças associacionais
- Igrejas
- Famílias
- Visitantes interessados na organização
